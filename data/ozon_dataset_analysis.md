## Prompt rules

- Title: до 200 символов по правилам Ozon, целевой потолок CardBot — 140 символов.
- Title structure: тип товара + бренд/модель при наличии + 2-4 важные характеристики.
- Description: 900-1400 символов.
- Description structure: 1 короткий абзац пользы + 3-5 преимуществ + сценарии применения.
- Hashtags: 8-15 штук, максимум 30.
- Characteristics: 6-12 строк в формате `Ключ: Значение, одна характеристика на строку`.
